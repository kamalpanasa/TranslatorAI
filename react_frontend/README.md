# React + Tailwind CSS Web Frontend

A modern, fast web application built using React (Vite) and styled with Tailwind CSS, designed to connect to the AeroTranslate API.

## Features
- **Modern User Experience**: Styled with an aesthetic dark mode layout and blur glassmorphism cards.
- **Tabs Interface**: Switch between Text translation, Document file parser uploaders, and speech processing.
- **Live Voice Recorder**: Records live audio from your microphone directly in the browser and uploads it.
- **Responsive Layout**: Works seamlessly across both mobile viewports and desktop browsers.

## Installation & Setup

1. **Install Node.js**:
   Make sure you have Node.js installed (LTS version recommended). You can download it from [nodejs.org](https://nodejs.org/).

2. **Navigate and Install Packages**:
   In your terminal, go to the React project directory:
   ```bash
   cd react_frontend
   npm install
   ```

3. **Start Development Server**:
   ```bash
   npm run dev
   ```
   Navigate to [http://localhost:3000](http://localhost:3000) (or the port shown in your terminal).

4. **Production Build**:
   To compile and compress the static assets for deployment:
   ```bash
   npm run build
   ```
   The compiled output will be written to the `dist/` directory.
